# Newsletter List Manager

A Laravel application for browsing email marketing lists and their subscribers via the SalesAutopilot API.

## Project Structure

| Path                | Description                          |
|---------------------|--------------------------------------|
| `app/`              | Backend logic (controllers, services)|
| `resources/views/`  | Blade frontend templates             |
| `docs/`             | AI chat log and project notes        |
| `docs/plans/`       | Project setup plans                  |
